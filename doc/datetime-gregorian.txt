datatime/gregorian.json
~~~~~~~~~~~~~~~~~~~~~~~

The file contains data on Gregorian calendar (or proleptic Gregorian
calendar).

* Structure

The file contains a JSON object with following name/value pair:

  month_days

    An array whose (/m/ - 1)th value is the number of days in month
    /m/
    <http://www.whatwg.org/specs/web-apps/current-work/#number-of-days-in-month-month-of-year-year>
    of a leap year.
